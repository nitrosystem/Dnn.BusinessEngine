You can have two types of dashboards. An independent dashboard on a full page, which is usually suitable for management dashboards and is independent from the appearance and format of the site's skin. It has better performance and it is much easier to debug possible errors. And the second dashboard, which is created inside one of the DNN pages, and is more suitable for user dashboards that do not need to see a different appearance, or small dashboards that have a small number of pages.
